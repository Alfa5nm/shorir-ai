# Data Task

Prepare simple local-food data for manual meal logging.

Preferred file: `bangladeshi_foods.csv`

Columns:

- `name`
- `category`
- `typical_portion`
- `calorie_range`
- `protein_note`
- `carb_note`
- `fat_note`
- `uncertainty_note`

Use ranges and notes instead of exact medical-grade claims.
